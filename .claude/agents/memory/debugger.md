# Memória do Debugger

## Especialização

- Debug de processo principal/renderer Electron
- Debug de componentes React
- Debug de FastAPI Python
- Debug de stores Zustand
- Problemas de comunicação IPC

## Abordagem

1. Reproduzir o problema
2. Coletar logs e mensagens de erro
3. Identificar causa raiz
4. Propor solução
5. Verificar que funciona

## Ferramentas

- DevTools do navegador para renderer
- Logging Electron
- Logging Python
- Vitest para testes frontend
- Pytest para testes backend

## Problemas Comuns

- Issues de state management (Zustand)
- Race conditions
- Erros de tipo TypeScript
- Erros de import
- Falhas de comunicação IPC
- Problemas de renderização de componentes
